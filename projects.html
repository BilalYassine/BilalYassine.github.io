 Sudoku Solver 

This is a GUI which generates a playable Sudoku board as well as being able to provide a solution to the given board. The program is written in Python and makes use of a backtracking algorithm in order to return a solution to any solvable game of Sudoku.

While building this project, I developed skills in Python, GUI development through tkinter, and backtracking algorithms.

 2D Platformer 

I am currently working with colleague and friend Robert DiSalle to create a 2D Action Platformer for PC built through Unity Game Engine. We are utilizing C# to script enemy AI, game events, etc. as well as Adobe Photoshop/Illustrator to design characters, maps, items, etc. The game follows a young wizard, banished from his home, who must master spells, solve puzzles, and conquer dungeons to redeem himself. We hope to finish and publish the game on Steam by the end of 2020.

This passion project has been instrumental in developing my skills for C#, Unity, ROS, Photoshop, Illustrator, and Git.

 Pokerbot 

I am creating an iOS app "Pokerbot" which will, with a given Texas Hold 'Em Poker hand, return a probability of winning for Pre-Flop, Flop, Run, and River, as well as compare the strength of two hands against each other in a "Heads Up" format. It will also allow you to log a history of your hands and total net profit.

Working on Pokerbot has helped me gain strong experience in Swift and iOS development.

 Quiz Creator GUI 

At the end of our third-level programming course at UW-Madison, we were assigned "teammates" and asked to create a group project which would demonstrate the programming concepts we had learned over the course of the semester. Several students and I decided to make a quiz creator. We used Java and JavaFX to display a quiz generated from inputted user data. Users could create and store questions for a quiz as well as take the quizzes themselves. We made use of a hash table back-end to store user data efficiently.

This Quiz Creator was my first experience developing a CS project as part of a group. This project was crucial in expanding my knowledge of Java, JavaFX, Git and data structures.
